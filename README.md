# React with TypeSctipt

TypeScriptでReactをかいていく練習

## 環境構築

- nodeのインストール

```
node --version
v12.14.1

npm --version
6.13.7
```

- yarnのインストール

```
npm install -g yarn

yarn --version
1.22.0
```

## プロジェクトを始める

``` BASH
# プロジェクトの作成
$ npx create-react-app my-react-ts-app --typescript
$ cd my-react-ts-app
$ npm install --save-dev tslint tslint-react tslint-config-prettier
# `tslint.json`を作成する
$ type nul > tslint.json
# 開発サーバーを
```
